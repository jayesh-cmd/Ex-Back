# Get Ex Back - WhatsApp Chat Analyser 💔 ➡️ 🥂

A fun, AI-powered FastAPI application that takes your exported WhatsApp chats and gives you relationship coaching in the style of a Gen-Z best friend. It analyzes where things went wrong, what your next move should be, and helps you craft the perfect follow-up text.

> **Disclaimer:** This is purely a FUN, experimental project powered by AI. It is NOT 100% correct, and the "advice" is generated by a language model. Do not take the analysis too seriously or use it as a substitute for real relationship counseling. Keep it lighthearted and enjoy the roast!

## Requirements

- Python 3.9+
- An Anthropic API Key (Claude Sonnet model access required)

## Setup

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

2. Create a `.env` file in the root directory and add your Anthropic API key:
```env
ANTHROPIC_API_KEY=your-anthropic-api-key
```

## How To Use

1. Start the backend server using Uvicorn:
```bash
python -m uvicorn main:app --reload
```
2. Open your browser and navigate to **[http://localhost:8000](http://localhost:8000)** to see the UI.
3. On your phone, open a WhatsApp chat, go to settings, and choose **Export Chat (Without Media)**.
4. Send the `.txt` file to your computer.
5. Upload the `.txt` file into the web UI drop zone and hit "Analyse Now" to generate your custom slides!
